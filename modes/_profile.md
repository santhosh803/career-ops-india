# Profile Customization

This file contains personal customizations for the career-ops system, specifically tailored for the Indian AI Startup market.

## Archetypes

When evaluating jobs, map them to these archetypes and use the corresponding framing strategies:

1. **AI/ML Engineer**: 
   - *Framing*: Emphasize end-to-end model training, deployment, and practical problem-solving using computer vision (YOLOv8) and LLMs.
   - *Proof Points*: Highlight the Smart Safety Sentinel project for CV, and real-world deployment experience.

2. **Applied AI / Gen AI Engineer**:
   - *Framing*: Focus on building practical GenAI applications, integrating models into existing systems (like ERPs), and utilizing RAG and ReAct agents.
   - *Proof Points*: Point to the ERP AI Procurement Assistant and AI SQL Data Analyst projects.

3. **LLM Engineer**:
   - *Framing*: Highlight specific experience with LangChain, local LLM inference (Ollama), prompt engineering, and grounding models via RAG (FAISS, ChromaDB).
   - *Proof Points*: Discuss the intricacies of the ERP AI Procurement Assistant and the ReAct pattern used in the AI SQL Data Analyst.

## Compensation Rules

- **Target Range**: ₹7 LPA to ₹30+ LPA
- **Currency**: INR (₹)
- **Minimum Acceptable**: ₹7 LPA
- **Negotiation Strategy**: Emphasize practical portfolio projects over years of experience. For startups, be open to ESOPs as part of the compensation package, but ensure base pay meets minimum requirements.

## Location Policy

- **Primary Preference**: Bengaluru
- **Secondary Preferences**: Chennai, Hyderabad
- **Remote**: Open to remote roles within India.
- **Scoring**: 
  - Bengaluru (On-site/Hybrid): +0.5
  - Chennai/Hyderabad (On-site/Hybrid): +0.2
  - Remote (India): +0.3
  - Other locations: -1.0

## Narrative & Positioning

- **The Pivot**: Position the background in ERP Implementation as a massive strength. "I understand how enterprises actually operate and map their data. Now I build AI systems that automate those exact workflows."
- **Focus**: Practical, deployed AI over theoretical research. "I build things that work, integrating AI into existing business logic."
